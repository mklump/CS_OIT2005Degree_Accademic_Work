Assignment 4 / Final project.

 

The class is divided into groups of 2 students each. The group is graded and each student in the group receives the same grade.

 

 

 


Description.

Students are required to write a compiler for a subset of the C language using bison and flex. The template flex and bison files are provided to the students. The compiler will produce code for the virtual machine described in the class.

 

The features of C that need not be implemented are described below. This implies that your compiler may issue an error when it sees the usage of these features or completely ignore it with the assumption that the instructor will not test it on your compiler.

 

Your compiler need not implement any error checking. Your compiler MUST not crash and MUST produce the correct code for a given C program.

 

C features that need not be supported.

  • Floating point numbers.
  • Strings.
  • Multi-dimensional arrays. Single dimensional arrays still must be supported.
  • No structures, unions or any aggregate data types.
  • No support for pointers ( This implies no syntax and semantic support ).

 

All the features of the C language must be supported in your compiler unless otherwise noted.
