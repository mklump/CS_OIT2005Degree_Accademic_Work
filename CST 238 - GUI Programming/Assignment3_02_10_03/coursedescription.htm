CST 238 - Course Outline

Instructor: Evelyn Wells, wellse@oit.edu

Web site: http://www.capital.ous.edu/~wellse/cst238/

Class meets: Mondays, 6-9:50pm and Saturdays (1/18, 2/1, 2/15, 3/1, 3/15), 8-11:50, all in Capital 1202
NOTE: no class on 1/20 (Martin Luther King Holiday)

Description (from the OIT catalog): Introduction to Windows based programming. Topics covered include a review of the standard user interface elements of Windows, the Windows Application Program Interface (API), message processing, writing Windows Procedures, working with text, using Windows resources, creating modal and modeless dialog boxes, and using the Graphics Device Interface. NOTE: I will also cover general user interface design and usability topics in the weekly lectures.

Prerequisite: CST 211 with grade "C" or better.

Required textbook: Programming Windows (1999), Charles Petzold, Microsoft Press.

Recommended textbook: Usability Engineering (1993), Jakob Nielsen, AP Professional.

Disabilities: Any student with a disability who anticipates a need for accommodation in this course is encouraged to talk with the instructor about his/her needs as soon as possible.

Grading: The final grade, which may range from A to F, is determined by a composite evaluation of the student's performance in:
35% - 2 Exams
40% - Assignments (5 worksheets, 8 labs, 8 reviews)
25% - Final Exam

Your grade will be calculated as follows:
90% + = A
80% + = B
70% + = C
60% + = D
60% - = F
Class participation will be considered in evaluating "borderline" grades.

Assignments: Assignments are due at the end of the class period, and will not be accepted late unless you have cleared it with me in advance. Late assignments will be penalized 20% per day, unless there are extreme extenuating circumstances. NOTE: Plagiarizing or copying will be grounds for immediate and automatic dismissal from the course.

Attendance: Students are expected to attend all class sessions and labs, and this is especially important for classes that meet only once a week. If you know you will be absent on a certain day, please inform me in advance so arrangements can be made to provide you with the material covered.
