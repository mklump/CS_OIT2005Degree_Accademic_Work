Managing strings

1. Read a set of names from a file named "names.dat" into an array. You may choose the actual path to the file. For example, the actual file path may be "c:\names.dat" or "c:\cst116\lab04\names.dat".

2. Allow the user to enter a name (called key) from the console.

3. Search for the name (key) the user entered in the array of names you have already built. If the key is found in the array, output to console the location at which the key was found. Let the user know if the key was not found.

4. Allow the user to search for as many keys as desired. The user will enter the special key "END" to denote she wants to end searching. Your program should end when the user enters this special key.
