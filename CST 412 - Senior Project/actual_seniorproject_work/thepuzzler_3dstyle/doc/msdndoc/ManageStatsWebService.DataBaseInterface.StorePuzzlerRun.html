DataBaseInterface.StorePuzzlerRun Method 

[This is preliminary documentation and subject to change.]

This web service will accept a complete six-item data row containing the next Puzzler - 3D Style Execution Run to keep track of.

[WebMethod(Description="Use this web service to save an execution set.", EnableSession=True, CacheDuration=0, BufferResponse=True, TransactionOption=TransactionOption.Disabled, TypeId=System.Web.Services.WebMethodAttribute)]
public string StorePuzzlerRun(
   string dictionary,
   string puzzle,
   string wordsFound,
   long dictionaryTime,
   long puzzleTime,
   long solutionTime
);

Parameters

dictionary
The dictionary for the current execution run.
puzzle
The puzzle for the current execution run.
wordsFound
The words that were found by the solution for the current execution run.
dictionaryTime
The time span taken to create the dictionary.
puzzleTime
The time span taken to create the puzzle.
solutionTime
The time span taken to execute the solution for the current run of everything.

Return Value

"Succeeded" if the the operation succeeded, and the error message if the operation failed.
