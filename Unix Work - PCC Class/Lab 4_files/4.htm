Chapter 10, Advanced File Processing:

  1. Assume you have a file, orders.dat, that contains the following data: 
      Mary  Donald  1020
Joe   Smith    348
Jack  Adams    912
Missy Jenkins  401
Jim   Donald  1034
    (the first column contains first names, the second column contains last names, and the third column contains quantities)

    For each of the following tasks, give a single command that will perform it:

    1. Sort the file by last name
    2. Sort the file by last name, then by first name
    3. Sort the file by quanity

    Hint: Use a text editor to create the file, and then test your answers.

  2. For each of the following, give a single command to find all lines in a file called customers.txt that:
    1. are empty or contain only spaces
    2. contain at least one A
    3. contain at least one A or a
    4. contain an uppercase or lowercase vowel
    5. do not contain an uppercase or lowercase vowel
    6. start with a Z
    7. do not start with a Z
    8. end with 000
    9. contain Unix or Linux
    10. contain Unix, followed by any number of other characters, followed by Linux

    Hint: Use a text editor to create a file that has each of the above, and then test your answers.

    Note: Do not use Extended Regular Expressions

  3. For each of the following tasks, give a single command that will perform it, regardless of your current working directory:
    1. List files that end in .html in the current directory and all subdirectories
    2. List all subdirectories of your home directory and all subdirectories
    3. Unconditionally delete (do not prompt for approval) all files and directories owned by you in /tmp and all subdirectories
    4. Conditionally delete (prompt for approval) all files and directories that are larger than 1 million bytes and owned by you in /tmp and all subdirectories

    Hint: Test your answers (be careful when testing: use echo rm instead of rm just to be safe).

Chapter 8, File Security

  1. For each of the following permissions, give the command to change permissions on a file, orders.txt, to those permissions, using a numeric mode argument:
    1. rw-r-----
    2. rwxr-x--x
    3. -ws--Sr-t

    Hint: Look at the man pages for the command to refresh your memory on what each of these symbols means.

  2. For each of the permissions from the previous question, restate the command using a symbolic mode argument.

  3. What command should I use if I wanted to grant (give, without changing any of the other existing file permissions) read and write permissions to just the group owner of a directory called /usr/lib/work, and all files and subdirectories thereof?

  4. How would I permanently change my shell configuration if I wanted all newly created files and directories to have the following permissions by default:
    • Files: rw-rw----
    • Dirs: rwxrwx--x

  5. What command should I use if I wanted to change the owner of a directory called data to the username charliet ?

  6. What command should I use if I wanted to change the group owner of a directory called data to the group acctg ?

  7. What command should I use if I wanted to change both the owner and group owner as specified in the previous two questions? Hint: See man pages.
