DLL demo

This example show how to create and use a Prolog DLL.  The example consists of two projects:

  • myDll - which produces a DLL myDll.dll
  • myDllUser - which produce an executable myDllUser.exe that uses myDll.dll

Both projects build the "target" in a common directory exe.

Build and Run

Load projects MyDll.prj6 and ../myDllUser/MyDllUser.prj6 into the VDE and press F9 (Execute), or build the project, and then run myDllUser.exe.

It is a good idea to run it from a command prompt (otherwise the window closes very rapidly).
