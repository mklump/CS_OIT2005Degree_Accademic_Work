PIE demo

This example demonstrates Prolog Inference Engine is a small Prolog Interpreter. It uses the standard Prolog syntax (K&M) and releases the most popular set of predicates.

Build and Run

Load the project Pie.prj6 into the VDE and press F9 (Execute), or build the project, and then run Exe/Pie.exe.

The program uses the message window for user input and outputs to the message window.
