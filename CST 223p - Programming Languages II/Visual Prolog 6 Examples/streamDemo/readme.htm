Stream demo

This example demonstrates VIP6 streams:

  • outputStream_string
  • outputStream_file
  • console::getConsoleOutputStream

It also illustrates that fact-databases can be written to any outputStream.

The reader is encouraged to extend the demo to cover inputStreams.

Build and Run

Load the project StreamDemo.prj6 into the VDE and press F9 (Execute), or build the project, and then run Exe/StreamDemo.exe.

The program writes to the console but it also produces a text file: Exe/myFile.txt

It is a good idea to run it from a command prompt (otherwise the window closes very rapidly).
