DESCRIPTION:

Students will create three proposals for possible senior projects.    The students and the instructor will then negotiate, refine and agree to one project proposal.  This proposal will be a contract between the student and the instructor defining what will be accomplished in the senior project series (CST 412, 422, 432).

OBJECTIVES:

This is the first of a four term sequence.  After this class you should be able to:

  • Create a project proposal

  • Understand the process for taking an software process from idea to proposal including:

  1. Defining the objective

  2.  Defining the features and high level requirements

  3. Defining the project context

  4. Define how you will know the project is successful.

  5.  Describing initial project risks

BOOKS:

No textbooks are required for this course, however, it is recommended that you purchase the textbooks for the 412/422/432 courses.

INDIVIDUAL WORK

This class is based on individual work.  There are no team assignments. During the senior project  you may help your fellow students in a limited way.  In general, others can help you by pointing out errors but they may not fix them.
