Pseudo Code Describing the Data Store Module and the Authentication Module

 

Data Store Module

 

A similar method will exist for each of the different objects and data members within those objects for all the data objects stored in our database.

 

Get{Object Type}{Member Type} (Object ID)

            Check database connection

            If database connection is not active

                        Then connect to database

                        If (connection can not be established)

                                    Then return database connection error

            Query Database for Object ID in the appropriate table

            If (Object ID doesn’t exist)

                        Then return error;

            Query Database for Member Type and Object ID

            If (there are no matches)

                        Then return “no entries” code

            Return results of query

 

 

Set{Object Type}{Member Type}(Object ID, Value)

            Check database connection

            If (database connection is not active)

                        Then connect to database

                        If (connection can not be established)

                                    Then return Database error

            Query Database for Object ID in the appropriate table

            If (Object ID doesn’t exist)

                        Then return error

            Send database command to change Member Type for Object ID to Value

            If (database error)

                        Then return database error

            Return success code


 

 

Authentication Module

 

NewSession (UserID)

            Check database connection

            If (database connection is not active)

                        Then connect to database

                        If (database connection error)

                        Then return database error

Generate SessionID

            Store SessionID, UserID, and ExpirationTime in database

            If (database error)

                        Then return database error

            Return SessionID

 

CheckSession (SessionID)

            Check database connection

            If (database connection is not active)

                        Then connect to database

If (database connection error)

                        Then return database error

            Query database for SessionID

            If (SessionID is not found)

                        Then return invalid sessionID error

            If (ExpirationTime is already passed)

                        Then return session timed out error

Else return UserID
